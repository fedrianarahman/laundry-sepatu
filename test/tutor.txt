1. git add . 
2. git commit -am"pesan"
3. git push origin master (mengirim pemburuan terbaru)
4. git pull origin master (mengunduh pembaruan terbaru) 


noted: 
langkah pertama setelah buka github, di pull dulu (kumpulkan data terbaru) 
===git pull origin master (mengunduh pembaruan terbaru) ===

kalo ada pembaharuan dari diri sendiri wajib di ketikin: 
===1. git add . ==
===2. git commit -am"pesan (Folder / File baru)"===
===3. git push origin master (mengirim pemburuan terbaru)==
